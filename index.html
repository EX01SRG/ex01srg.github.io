SunPi GeoStation

The SunPi GeoStation is a compact, solar-powered, battery-backed Raspberry Pi system designed for hosting and serving geospatial applications. The unit features cloud-based connectivity via Cloudflare, supporting advanced geospatial data analysis with PostGIS and intuitive data visualization with TerriaJS.

Hardware Components

  • Raspberry Pi: High-performance computing in a compact form, pre-loaded with a Linux-based operating system.
  • Solar Power Module: Highly efficient solar panel array, optimal performance in varying sunlight conditions, capable of continuous operation during daylight hours.
  • LiFePO4 Battery System: Built-in Lithium Iron Phosphate (LiFePO4) battery for reliable energy storage, ensuring uninterrupted operation during non-sunlight hours.
  • Power Management System: A smart system to switch between solar power and battery reserves, and manage the charging from the solar module. It safeguards the battery from overcharge and deep discharge states.

Software Components

  • PostGIS: Open-source spatial database extender for PostgreSQL, supports geographic objects allowing location queries in SQL.
  • TerriaJS: JavaScript library for creating 3D geospatial data visualization applications. Supports multiple data formats and services.
  • Cloudflare Integration: Ensures fast, secure, and reliable global network performance. Offers protection from DDoS attacks and optimizes software delivery.

Connectivity

Cloud Connectivity: The device supports Wi-Fi and Ethernet connectivity, ensuring seamless communication with the cloud. It allows users to access, interact with, and share geospatial data securely over the network.
